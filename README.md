# 🌐 Gamified E-Commerce & Virtual Mall Platform

> **"เปลี่ยนประสบการณ์การช้อปปิ้งออนไลน์แบบเดิม ๆ ให้กลายเป็นโลกเสมือนจริงที่ทุกคนสามารถเป็นเจ้าของธุรกิจได้"**

---

## 🎯 Product Vision (วิสัยทัศน์ของโปรเจกต์)
แพลตฟอร์มนี้ไม่ใช่แค่เกมสร้างเมืองทั่วไป แต่คือ **Gamified E-Commerce Store** ในรูปแบบ **Virtual Mall (ห้างสรรพสินค้าจำลอง 3D แบบ Multi-vendor)** ที่เชื่อมต่อโลกแห่งการเล่นเกมเข้ากับระบบเศรษฐกิจและการซื้อขายสินค้าจริงในชีวิตประจำวัน 
วิสัยทัศน์ของเราคือการทลายกำแพงการช้อปปิ้งที่น่าเบื่อบนหน้าเว็บ 2D ให้กลายเป็นการเดินเลือกซื้อสินค้าในเมืองจำลอง 3D ที่สดใส มีชีวิตชีวา โดยเปิดโอกาสให้บุคคลทั่วไป ร้านค้า และแบรนด์ต่าง ๆ สามารถเข้ามา **"เช่าพื้นที่"** เปิดร้านค้าของตนเอง ผูกระบบหลังบ้าน ขนส่ง และชำระเงินจริง รวมถึงมี **AI Agent NPC** ประจำร้านที่คอยต้อนรับและปิดการขายให้กับลูกค้าได้ตลอด 24 ชั่วโมง โดยตัวแพลตฟอร์มรองรับตลาดระดับสากลอย่างไร้พรมแดน

---

## 💰 Revenue Model (โมเดลสร้างรายได้หลัก)
1. **Space Subscription (ค่าเช่าพื้นที่รายเดือน):** เก็บค่าเช่าจากร้านค้าจริงที่ต้องการมาตั้งบู้ทหรือสร้างตึกในเกม โดยแบ่งตามทำเลและโซนพื้นที่ (Zoning System)
2. **In-game Advertising (พื้นที่โฆษณา):** โมเดลขายป้ายโฆษณา (Billboard) ตามจุดต่าง ๆ ในเมือง ดึงรูปภาพโฆษณาสดจากระบบ Web API 
3. **Transaction Fee (ค่าธรรมเนียม):** เก็บเปอร์เซ็นต์ส่วนแบ่งขนาดเล็กจากการซื้อขายสินค้าจริงที่เกิดขึ้นผ่านระบบ AI Agent ภายในเกม

---

## 🗺️ Platform Features & 7 Languages
* **Zoning System:** การแบ่งผังเมืองจำลองออกเป็นโซนทวีป/ประเทศ เพื่อจัดระเบียบร้านค้า
* **Dynamic Billboard Ad System:** ระบบป้ายโฆษณาอัจฉริยะที่ดึงข้อมูลภาพแบนเนอร์จากผู้เช่าโฆษณาจริงมาแสดงผลในเกมแบบ Real-time
* **AI Agent Storefront:** NPC ประจำร้านค้าที่ขับเคลื่อนด้วย AI สามารถแนะนำสินค้า แจกคูปอง และปิดการขายกับลูกค้าได้
* **Global Multi-language Support:** รองรับ 7 ภาษาหลักทั่วโลก: **อังกฤษ (EN), ไทย (TH), จีน (ZH), ญี่ปุ่น (JA), เกาหลี (KO), เวียดนาม (VI) และ อาหรับ (AR)** (รวมถึงระบบอักษรจากขวาไปซ้าย - RTL)

## 🛠️ Technology Stack
* **Game Engine:** Unity (C#) - Universal Render Pipeline (URP)
* **Localization System:** Unity Localization Package + TextMeshPro (TMP)
* **Backend & Database:** Firebase / Supabase 
* **API Connection:** HTTP Web Requests / JSON Parsing
